Water Habit Tracker — Веб-приложение для водного баланса
Современное веб-приложение на базе Flask для отслеживания ежедневного потребления воды и ведения серии дней здоровой привычки. Проект позволяет пользователям отмечать выпитую воду через стильный веб-интерфейс, автоматически сохраняя и обновляя результаты в режиме реального времени.

Используемые технологии:
Python — основной язык программирования (логика сервера).
Flask — легкий веб-фреймворк для создания серверной части и обработки HTTP-запросов.
Jinja2 — встроенный во Flask шаблонизатор для динамического вывода данных в HTML.
HTML/CSS — адаптивная верстка интерфейса с градиентным фоном и анимацией кнопок.
JSON — встроенная текстовая база данных для постоянного хранения результатов.
Структура проекта:
коды по пайтону/
│
├── templates/
│   └── index.html         # Веб-интерфейс приложения (HTML + CSS стили)
│
├── pro.py                 # Основной серверный код (Flask-приложение)
├── water_tracker.json     # База данных проекта в формате JSON (состояние трекера)
├── water_data.json        # Дополнительный лог дат активности в формате JSON
└── README.md              # Документация проекта

1. Установка Flask
Для работы веб-интерфейса необходимо установить микрофреймворк Flask. Откройте терминал в VS Code и выполните команду:
<pip install flask>
2. Проверка установки
Чтобы убедиться, что фреймворк успешно установлен, проверьте его версию командой:
<python -m flask --version>
3. Запуск веб-приложения
Запустите файл pro.py, который отвечает за работу веб-сервера и визуальной части проекта:
<python pro.py>
4. Переход на сайт
После успешного запуска в терминале появится строка:
<* Running on http://127.0.0.1:5000>

Зажмите клавишу Ctrl и кликните левой кнопкой мыши (Ctrl+ЛКМ) по ссылке, либо скопируйте адрес <http://127.0.0.1:5000> в любой браузер.
